# 总结：

1.技术选型：单独制作移动端页面方案，流式布局

2.搭建项目结构

3.设置视口标签及引入初始化样式：normalize.css,index.css,<meta ...>

3.具体布局：二倍精灵图（整体缩放，再测量具体位置，左移上移为负），搜索栏（固定定位 fixed），品牌日（去除图片顶部缝隙，图片左上及右上圆角），快报（第2、3张图片加左边框）
